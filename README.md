# Portfolio – Side projekt

## 📌 Overview

This portfolio is a collection of my school projects that I'm very proud of. This project is an open book to how I designed and constructed my project from scratch and how that gained momentum.

Open it here:  
👉https://avdrxxa.github.io/portfolio/


## 🎯 What It Is

This project is a begginer portfolio that showcases my school and side projects.

This project needed only HTML, CSS and a little bit of JavaScript, just to give it a little personality and sass.


## 🧠 What I Learned

While building this project, I learned:

How to use JavaScript to manipulate the DOM

emailTo function in JavaScript

Handling user input and events (clicks, buttons)

Structuring a project and organizing code

Improving problem-solving skills through debugging continuous


## 🛠️ Technologies Used

HTML – for structuring the web page  

CSS – for styling and layout  

JavaScript – for functionality and interactivity  

GitHub Pages- for deploying the page 


## 🚀 Features

Collection of web and graphic projects

About, projects and contact pages 

Clear navigation system 

Buttons for user actions (submit, browse through menu, etc.)

Design avaliable in desktop and mobile

Simple and clean user interface
